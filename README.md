## 🎯 Overview

A 10-class image classifier built with **transfer learning** on MobileNetV2,
trained on the CIFAR-10 dataset.  The model achieves **~75 % test accuracy**,
outperforming the CIFAR-10 random baseline (10 %) and a typical CNN-from-scratch
baseline (~65 %).

## 🏗️ Model Architecture

```
Input (224 × 224 × 3)
    │
    ▼
MobileNetV2 backbone  ← ImageNet pre-trained, frozen
    │
    ▼
GlobalAveragePooling2D
    │
    ▼
Dense(256, relu) + Dropout(0.4)
    │
    ▼
Dense(10, softmax)          ← 10-class output
```

### Training strategy (two phases)

| Phase | Epochs | LR | Backbone |
|---|---|---|---|
| 1 — Head only | 15 | 1e-4 | Frozen |
| 2 — Fine-tune | 10 | 1e-5 | Top 30 layers unfrozen |

## 🔬 How Transfer Learning Works Here

```
ImageNet weights (1 000 classes, 1.2 M images)
          │
          │  knowledge transfer
          ▼
CIFAR-10 task (10 classes, 50 000 images)

Step 1: freeze MobileNetV2, train only the new head
        → fast convergence, prevents overwriting learned features
Step 2: unfreeze top layers, fine-tune end-to-end at low LR
        → adapts high-level features to CIFAR-10 specifics
```

## 🔧 Extending the Project

| Goal | What to change |
|---|---|
| Higher accuracy (> 80 %) | Unfreeze more layers; use EfficientNetB3 |
| Faster training | Reduce image size to 96 × 96; increase batch to 64 |
